# 어린 왕자의 여행 문제 해결하기

어린 왕자가 행성계를 여행하면서 최소한으로 행성계 경계를 진입/이탈하는 횟수를 계산하는 프로그램을 Java로 구현한다.

## 프로그램 구조

프로그램은 다음과 같은 메인 컴포넌트로 구성된다:

- `main` 메서드: 입력을 받고 각 테스트 케이스에 대해 처리를 수행한다.
- `isInCircle` 메서드: 주어진 점이 원 안에 있는지 여부를 계산한다.

### 메인 메서드

`main` 메서드는 다음 단계로 처리를 수행한다:

1. 테스트 케이스의 수 `T`를 입력받는다.
2. 각 테스트 케이스에 대해:
    - 출발점 `(x1, y1)`과 도착점 `(x2, y2)`을 입력받는다.
    - 행성계의 수 `n`을 입력받고, 각 행성계에 대해 중심 `(cx, cy)`과 반지름 `r`을 입력받는다.
    - 출발점과 도착점이 각 행성계 경계 내외부에 있는지 판단하여 진입/이탈 필요성을 계산한다.
    - 진입/이탈 횟수를 출력한다.

### isInCircle 메서드

`isInCircle` 메서드는 다음을 수행한다:

- 점 `(x, y)`가 원 `(cx, cy)` 반지름 `r`에 속하는지 여부를 계산한다.
- 원의 방정식 `(x-cx)^2 + (y-cy)^2 < r^2`을 사용하여 계산한다.
- 원 내부에 있으면 `true`, 아니면 `false`를 반환한다.
